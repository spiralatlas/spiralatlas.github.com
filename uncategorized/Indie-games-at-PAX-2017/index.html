Indie games at PAX 2017

27th October 2017 #from tumblr #other people's games #recommendations #pax #conventions

Indie Games I checked out that look cool:

  • The Gardens Between Pretty puzzle game with an interesting time-control mechanic
  • Intergalactic Space Princess Cute adventure game
  • Qinoto Cute puzzle game where you “fold” the image on the screen to make a new image.
  • Necrobarista edgy animated visual novel about a necromancer
  • Morphite Space exploration game with plot.
  • The Eyes of Ara Pretty fantasy point and click puzzle and exploration game
  • Ticket to Earth turn based fighter using match-3 type mechanics.
  • Winter’s Wake a fantasy text adventure with an interesting movement mechanic.
  • Rainbow Reef Pretty reef building game

Games that looked cool but I didn’t get a chance to play:

  • Earthlight VR space station exploration
  • Rumu Narrative driven adventure game about a robot vacuum cleaner
